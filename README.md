# 📱 Mobile Paradise – Online Mobile Shopping Website

An e-commerce web application for browsing, selecting, and purchasing mobile phones, built using a full-stack approach with frontend, backend, and database integration.

---

## 🚀 Overview

**Mobile Paradise** is an online shopping platform that allows users to:
- Browse different mobile phones  
- View product specifications  
- Add items to cart  
- Place orders and track them  

This project demonstrates end-to-end web development including UI design, backend logic, and database management.

---

## 🧠 Key Features

- 🛍️ **Product Listing** – Browse available mobile phones  
- 📄 **Product Details** – View specifications and pricing  
- 🛒 **Shopping Cart** – Add/remove items  
- 📦 **Order Management** – Place and view orders  
- 👤 **User Interaction** – Simple and intuitive navigation  
- 💻 **Responsive UI** – Clean interface using HTML, CSS, JS  

---

## 🏗️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: PostgreSQL  

---

## 🧪 How It Works

1. User browses available mobile products  
2. Selects a product and views details  
3. Adds product to cart  
4. Places an order  
5. Order details are stored in the database
